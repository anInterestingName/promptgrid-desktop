# Product Brief

## Name

FangCun / 方寸画境

## One-Line Positioning

A local-first AI visual exploration desktop workspace that turns one creative idea into a grid of editable image directions.

## Target Users

Initial target users:

- Designers exploring visual directions.
- Self-media creators making cover images and article visuals.
- E-commerce operators exploring product scene and campaign images.

Later users:

- Game/IP creators.
- Ad operators.
- Small creative teams.

## Core Workflow

```text
Create project
-> enter original prompt
-> choose style, aspect ratio, quality
-> analyze prompt into 9 variants
-> edit variant prompts
-> generate 3x3 grid
-> preview images
-> retry failed cells
-> regenerate weak cells
-> expand from strong cells
-> export results
```

## Differentiation

The product should not compete only on batch generation.

The differentiator is:

- AI-assisted prompt branching.
- Grid-first comparison.
- Cell-level regeneration.
- Cell-to-grid expansion.
- Local project history and asset ownership.

## MVP Features

Required:

- Local project creation.
- Original prompt input.
- 9 prompt variants.
- Prompt variant editing.
- 3x3 image task grid.
- Task state per cell.
- Retry failed cell.
- Regenerate completed cell.
- Expand selected cell into a new 3x3 round.
- Single image preview.
- Single image export.
- Grid export.
- Local history.

Deferred:

- 4x4 and 5x5 grids.
- CSV import.
- Reference image workflows.
- Local model deployment.
- Cloud sync.
- Accounts and payments.

## Product Boundaries

Phase 1:

- Desktop only.
- Local data only.
- User-provided AI API key.

Future:

- Go cloud backend.
- Mini program.
- Mobile app.
- Web version.
- Cloud sync.
- Team library.
